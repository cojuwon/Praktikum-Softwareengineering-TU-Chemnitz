# Requirement #: FR-04

# Requirement Type: Functional

# Event/BUC/PUC #:
- Nutzer:in möchte gespeicherte Daten nach bestimmten Kriterien auswerten

# Description:
- das System ermöglicht die Filterung gespeicherter Daten nach frei wählbaren Kriterien (z. B. Zeitraum, Geschlecht, Beratungsart)
- gefilterte Ergebnisse werden als Statistik angezeigt und können als PDF-, XLSX- oder CSV-Datei exportiert werden
- Nutzer:innen können individuelle Filtereinstellungen als „Presets“ speichern, laden und löschen
- Nutzer:innen mit erweiterten Rechten (Erweiterung, Admin) können zusätzlich geteilte Presets erstellen und verwalten
- drei Standard-Presets sind im System vordefiniert

# Rationale:
- die regelmäßige Erstellung von Statistiken ist eine zentrale Anforderung der Beratungsstellen
- eine automatisierte und flexible Statistikausgabe reduziert manuellen Aufwand und minimiert Übertragungsfehler

# Originator:
- Mitarbeiter:innen des Bellis e.V.

# Fit Criterion:
- Nutzer:innen können Daten erfolgreich nach mindestens drei Kriterien filtern (z. B. Zeitraum, Beratungsart, Geschlecht)
- Ergebnisse lassen sich in den Formaten PDF, XLSX und CSV ohne Datenverlust exportieren
- eigene Presets können gespeichert, geladen und gelöscht werden
- Standard-Presets sind verfügbar
