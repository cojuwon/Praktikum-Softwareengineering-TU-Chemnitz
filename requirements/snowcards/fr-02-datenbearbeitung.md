# Requirement #: FR-02

# Requirement Type: Functional

# Event/BUC/PUC #: 
- Event: Eine gespeicherte Anfrage oder ein Fall wird zur Bearbeitung geöffnet
- BUC: Mitarbeiter:in bearbeitet bestehende Anfragen oder Beratungsfälle
- PUC: Mitarbeiter:in sucht Datensatz, nimmt Änderungen vor und speichert aktualisierte Informationen

# Description:
Das System muss es allen berechtigten Mitarbeiter:innen ermöglichen bestehende Datensätze:
- zu suchen
- anzuzeigen
- zu bearbeiten
- zu aktualisieren

# Rationale:
- im Beratungsprozess werden Fälle fortlaufend ergänzt
- eine Bearbeitungsfunktion ist notwendig, um Änderungen und neue Informationen nachvollziehbar zu dokumentieren

# Originator:
- Mitarbeiter:innen des Bellis e.V.

# Fit Criterion:
- berechtigte Nutzer:innen können Datensätze über definierte Suchkriterien (z. B. Datum, ID, Name) finden 
- nach Änderungen kann der Datensatz erfolgreich gespeichert werden
- die letzte Änderung wird mit Datum und Nutzer:in im System vermerkt  
- Aktualisierung von Beratungsterminen (Datum, Art, Status) ist möglich
